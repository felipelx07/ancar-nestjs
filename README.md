# Ancar  - NestJS + React APP


## Descrição

Este exercício tem como objetivo implementar um projeto de elaboração de
enquetes simplificadas, onde um usuário pode criar sua enquete e associar perguntas e
respostas a elas.

## Instalação

```bash
$ yarn install
```

## Executando o App

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Exeutando Testes

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
